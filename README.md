# MHW Builder @ https://mhw-builder.vercel.app/

Next.js + MUI project, based on [this example](https://github.com/mui/material-ui/tree/master/examples/nextjs).

### Requirements
- Node
- npm

### Build & Run

```
npm install
npm run dev
```

Finally, open http://localhost:3000
